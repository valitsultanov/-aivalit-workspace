# PROJECT HUB (Single Source of Truth)

Этот файл — главный индекс всех проектов.

## Как работаем
1. Ты отправляешь аудио с новым проектом.
2. Я расшифровываю и создаю папку проекта.
3. Заполняю `PROJECT.md` + `ROADMAP.md` + `TASKS.md`.
4. Добавляю проект в этот индекс.
5. Ключевые решения дублирую в `MEMORY.md` (main session continuity).
