# RELEASE CHECKLIST

## Перед релизом
- [ ] `git status` чистый
- [ ] Все изменения закоммичены понятными сообщениями
- [ ] Проверены чувствительные данные (нет токенов/секретов)
- [ ] Проверены крупные файлы (нет >100MB)

## Публикация
- [ ] Обновлен `main` (или PR из `dev`)
- [ ] `git push`
- [ ] Проверка репо на GitHub

## После релиза
- [ ] Запись в `memory/YYYY-MM-DD.md`
- [ ] Обновлен `docs/ROADMAP.md` (если изменились приоритеты)
