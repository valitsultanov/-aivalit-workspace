# Что нужно прислать, чтобы бот изучил канал и комментарии

Чтобы собрать точный FAQ и карту интентов из реальных диалогов, нужен экспорт/выгрузка:

## Вариант A (лучший)
- Экспорт Telegram (Desktop) постов + комментариев канала за 30-90 дней
- Формат: JSON/HTML/TXT

## Вариант B
- 30-50 ссылок на ключевые посты
- 100-200 самых частых комментариев/вопросов (текстом)

## Вариант C
- Добавить бота в админы канала + права чтения комментариев (если используете связку с обсуждением)

После получения данных:
1) Кластеризация интентов
2) Топ-20 FAQ
3) Карта возражений
4) Готовые ответы/скрипты для бота
