# Prompt Base — Beginner Checklist (на основе OpenAI best practices)

## Формула новичка (самая рабочая)
1. **Инструкция в начале**: что сделать
2. **Контекст**: кто ты, для кого, какая цель
3. **Формат ответа**: список / таблица / шаги
4. **Ограничения**: длина, стиль, язык
5. **Критерий качества**: что считать хорошим результатом

---

## Шаблон промпта

```text
Сделай [задача] для [аудитория].

Контекст:
- Я: [кто ты]
- Ниша: [ниша]
- Цель: [лиды/продажи/охваты]

Формат ответа:
- [например: 7 пунктов + CTA]

Ограничения:
- Язык: русский
- Длина: до [N] символов
- Стиль: [простой/экспертный]

Критерий качества:
- конкретно, без воды, с примерами
```

---

## 5 правил для новичка
- Пиши конкретно, а не «сделай красиво»
- Всегда задавай формат ответа
- Сначала zero-shot, потом добавляй примеры (few-shot)
- Не проси «не делай Х», лучше пиши «сделай Y»
- Улучшай итерациями: черновик → доработка

---

## Пример (распаковка себя)

```text
Помоги мне выбрать 3 направления в AI, где я могу быстро получить результат за 7 дней.

Контекст:
- Чем я занимаюсь: [текст]
- Что мне нравится: [текст]
- Сколько времени в день: [текст]

Формат ответа:
1) 3 направления
2) Для каждого: что делать, какой результат, план на 7 дней
3) Первый шаг на сегодня (30 минут)

Ограничения:
- Пиши простым языком
- Без сложных терминов
```
