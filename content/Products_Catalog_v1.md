# Products Catalog v1 — AiValit

## 1) Промпт-книга для начинающих
**Для кого:** новички в AI, начинающие креаторы/эксперты
**Результат:** быстрый старт в промптинге, шаблоны для ежедневных задач
**Когда предлагать:** пользователь «не знает с чего начать», просит готовые примеры
**Формат:** материал/гайд
**CTA:** «Отправить промпт-книгу и мини-план внедрения на 3 дня?»

## 2) Банановый экспресс (интенсив по нейросетям, в записи)
**Для кого:** те, кому нужен быстрый прикладной результат
**Модули (актуальные):**
1. Визуал для маркетинга
2. Инфографика + карточки товаров
**Планируются:**
3. Нейросети для инфобизнеса (контент для соцсетей)
4. Коммерческая иллюстрация (персонажи, комиксы, книги)
**Когда предлагать:** запрос «нужен быстрый результат/система/структура»
**CTA:** «Показать, какой модуль тебе даст максимальный результат первым?»

## 3) Личное ведение Валита Султанова
**Для кого:** предприниматели, эксперты, команды
**Результат:** персональная AI-система под задачи бизнеса
**Что включает:**
- настройка системы работы с AI
- настройка оплат AI-сервисов
- оптимизация затрат через AI
**Когда предлагать:** запрос на внедрение, масштабирование, экономию
**CTA:** «Хочешь, оценю твой кейс и предложу формат личного ведения?»
